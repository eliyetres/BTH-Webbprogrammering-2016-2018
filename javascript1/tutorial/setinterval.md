Set Interval
==============================



```
/**
 * Execute a function in intervals (milliseconds)
 **/
var myInterval = setInterval(function() {
	//code to be executed
        }, 2000);

/**
 * Stop the interval timer
 * setInterval must be assigned to a variable
 **/
 clearInterval(myInterval);

```
